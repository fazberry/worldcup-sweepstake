# Worldcup Sweepstake

Create json object with the sweepstake participants

Just create a json file with your participants details:

````
[{
    "code": "RUS",
    "participant": {
        "country" : "Russia",
    	"name" : "Ben",
    	"avatar" : "URL"
    }
},
{
    "code": "KSA",
    "participant": {
        "country" : "Saudi Arabia",
    	"name" : "Jenna",
    	"avatar" : "URL"
    }
},
{
    "code": "EGY",
    "participant": {
        "country" : "Eygpt",
    	"name" : "Paul",
    	"avatar" : "URL"
    }
}

````
